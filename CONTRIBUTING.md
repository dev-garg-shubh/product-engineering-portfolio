# Contributing

Thanks for improving Product Engineering Portfolio.

## Development

1. Fork the repository and create a focused branch.
2. Install dependencies with `pnpm install`.
3. Run the local app with `pnpm run dev`.
4. Keep reusable behavior in components and personal content in configuration.
5. Run `pnpm build` before opening a pull request.

## Pull Requests

- Explain the user-facing change and its motivation.
- Include screenshots for visual changes in light and dark modes.
- Check desktop and mobile layouts.
- Avoid committing personal credentials, private CVs, or client data.
- Keep unrelated formatting and refactors out of the change.

By contributing, you agree that your work will be licensed under the MIT License.
